# Water Reminder App

A Flutter application designed to help users track their daily water intake, set personalized goals, and receive timely reminders to stay hydrated.

## Features

- **Daily Water Intake Tracking:** Log your water consumption with predefined amounts or custom entries.
- **Personalized Daily Goal:** Set and adjust your daily water intake target.
- **Progress Visualization:** A circular progress indicator on the home screen shows your real-time progress towards your daily goal.
- **Water Intake History:** View a detailed history of your water consumption.
- **Local Notifications:** Receive reminders to drink water throughout the day (future implementation).
- **Settings:** Configure your daily goal and notification preferences.

## Installation

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Flutter SDK (version 3.35.7 or higher)
- Dart SDK (version 3.9.2 or higher)

### Setup

1. Clone the repository:
   ```bash
   git clone git@github.com:ijaz-at-studio93/water_reminder.git
   cd water_reminder
   ```
2. Get dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```

## Usage

- **Home Screen:** Log your water intake, view your daily progress.
- **History Screen:** See a list of all your logged water intake.
- **Settings Screen:** Adjust your daily water goal.
